## 문제1
C언어에서 프로그램은 토큰(token)의 나열로 구성되는 데, 토큰과 토큰 사이의 공백은 무시된다.
C언어는 다음과 같이 6가지 토큰으로 구성된다.

- 식별자(identifier)
- 키워드(keyword)
- 상수(constant)
- 문자열(string)
- 연산자(operator)
- 구분자(punctuator)

1) 토큰이 무엇인지 설명하고(1점),<br />
2) 상기 6개 토큰 종류에 대해서 설명하고(2점),<br />
3) 다음 프로그램의 토큰을 6개의 구분으로 분류하라(2점).

```c
int main(){
  int a = 0;
  
  printf("%d\n", a);
}
```
